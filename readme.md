# example.grpc.dev

## usage

- up

```sh
docker-compose up
```

- down

```sh
docker-compose down
```

- build

```sh
docker-compose up --build
```

- start

```sh
docker-compose start
```

- stop

```sh
docker-compose stop
```

- view document

```sh
docker-compose exec protoc-go npm start
```

- run api

```sh
docker-compose exec api go run ./
```

- run client

```sh
docker-compose exec client go run ./
```
